Web Application "LIBAT"

A scalable ERP backend built with Laravel and MySQL

description Project Overview

Libat is an ERP-based application developed to support the recording and management of ship pilotage operations at PT. Biru Arnawama Timur. The system is designed to integrate various administrative processes such as Work Order (SPK) creation, Form A21 Captain, pilotage realization, and RPKRO submission into a single structured platform.

Libat also provides document automation features such as invoice and pranota generation in PDF format, as well as data recapitulation in Excel to support reporting needs. With integration to the Ministry of Transportation system via XML SOAP API such as Inaport / Inaops, this application ensures accurate and real-time data synchronization, thereby improving operational efficiency and productivity
