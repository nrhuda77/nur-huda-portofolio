Pelita Air Trade Union

A scalable ERP backend built with Laravel and MySQL

description Project Overview

SPPAS (Serikat Pekerja Pelita Air Service) is a web-based application developed to support labor union activities in a digital and integrated manner. The application provides a platform for members to access information, submit aspirations, and participate in various organizational activities more easily and transparently.

SPPAS features a landing page as an information and recruitment medium, along with an authentication system that allows members to access core functionalities. Members can submit reports regarding inappropriate policies, attach evidence such as images, videos, and documents, and monitor financial transparency as well as organizational agendas.

Technically, the application is built using Laravel and MySQL, with email notification support for important activities. The system also supports document generation and distribution in PDF format, including employment agreements that can be automatically watermarked with the member’s name. In addition, financial reports can be managed using uploaded Excel templates, simplifying administrative and reporting processes
